Role des Composants

-Ecran : Appareil périphérique externe qui permet de visualiser l’interface de l'ordinateur.

-Clavier : Appareil périphérique externe qui permet d'insérer des caractère ou de contrôler les opérations de l’ordinateur par le biais de touches.



-Souris : Appareil périphérique externe permettant de déplacer un indicateur sur l'écran et de sélectionner et agir sur les éléments des tâches.



-Port USB : Port permettant de connecter un appareil périphérique externe via câble USB.

-Carte graphique : Appareil de traitement qui sert à créer des formes géométriques.

-Carte mère : La carte mère est le composant principal de l'unité centrale dont le rôle est de centraliser et traiter les données échangées dans un ordinateur à l'aide du processeur, qui est fixé dessus. La carte mère gère donc entre autres le disque dur, le clavier et la souris, les ports USB...

-Mémoire RAM : Signifiant Random Access Memory en anglais, la mémoire RAM ou mémoire vive permet à tout ordinateur de stocker des informations provisoires.
