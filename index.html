Role des Composants

-Ecran : Appareil périphérique externe qui permet de visualiser l’interface de l'ordinateur.

-Clavier : Appareil périphérique externe qui permet d'insérer des caractère ou de contrôler les opérations de l’ordinateur par le biais de touches.

-Souris : Appareil périphérique externe permettant de déplacer un indicateur sur l'écran et de sélectionner et agir sur les éléments des tâches.

-Port USB : Port permettant de connecter un appareil périphérique externe via câble USB.

-Carte graphique : Appareil de traitement qui sert à créer des formes géométriques, et à procéder des images.

-Disque dur : Appareil périphérique interne ou externe qui permet d'écrire/conserver de l’information sous forme de mémoire morte ou ROM, Read Only Memory en anglais.

-Processeur : Appareil centrale de l’ordinateur branché directement dans la carte mère, il traite l’information en accomplissant les calculs et les taches de celui-ci. Il fait cela grâce à des transistors.

-Port HDMI : Signifiant High-Definition Multimedia Interface en anglais, le port HDMI est un port permettant de connecter plusieurs appareils de haute définition entre eux, par exemple connecter une console de jeu à un ordinateur.

-Carte mère : La carte mère est le composant principal de l'unité centrale dont le rôle est de centraliser et traiter les données échangées dans un ordinateur à l'aide du processeur, qui est fixé dessus. La carte mère gère donc entre autres le disque dur, le clavier et la souris, les ports USB...

-Mémoire RAM : Signifiant Random Access Memory en anglais, la mémoire RAM ou mémoire vive permet à tout ordinateur de stocker des informations provisoires.

-Bios : Basic Output Input System en anglais, le BIOS sont le système de contrôle qui permet d'effectuer des opérations de bases tel que le contrôle des éléments de l’ordinateur branchés dans la carte mère. Le BIOS se met en marche à chaque démarrage de l’ordinateur.
